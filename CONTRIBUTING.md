# Contributing to AI Avatar Chatbot

Thank you for your interest in contributing!

## How to Contribute

### Adding a New Module

1. Follow the modular architecture in ARCHITECTURE.md
2. Create self-contained modules
3. Use AppContext for frontend communication
4. Document your module's API

### Code Style

#### Backend
- Follow PEP 8
- Use type hints
- Add docstrings
- Use Pydantic for validation

#### Frontend
- Use functional components
- Follow React best practices
- Use Material-UI components
- Keep components small and focused

### Testing

Before submitting:
1. Test your changes locally
2. Check for console errors
3. Verify API endpoints work
4. Test with both Gemini and OpenAI (if available)

### Pull Request Process

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test thoroughly
5. Submit a pull request

### Module Guidelines

Each module should:
- Be independent and self-contained
- Have its own README
- Follow existing patterns
- Not break existing modules

## Questions?

Open an issue for:
- Bug reports
- Feature requests
- Questions about architecture
- Help with implementation

Thank you for contributing!
